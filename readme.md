## アプリケーション名

「terawork」

## アプリケーション概要

エンジニア向けの求人サイトです。

## URL

https://terawork.herokuapp.com

## 使用技術

- HTML/CSS
- PHP 7.4
- Laravel 5.8
- MySQL 8.0

## 機能一覧

- 登録・ログイン機能(マルチ認証)
    - Googleアカウントで登録/ログイン
    - Githubアカウントで登録/ログイン
- 応募者
    - プロフィール機能
        - 資格
        - 趣味
        - 自己紹介 
    - 応募履歴機能
    - お気に入り機能
    - 閲覧履歴機能
- 掲載者
    - 求人リストのCRUD機能
- 求人検索機能
    - 都道府県・月収・言語・フリーワードで検索可能
- ページネーション機能(Bootstrap)
